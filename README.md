### 🚀 Featured Projects

* **[Mental Health Trajectory Predictor](https://github.com/hari047/Mental-Health-Research-Project)** - A machine learning framework designed to predict longitudinal recovery trajectories (Improved, Worsened, Stable) from digital mental health discussion threads. Extracted early-stage linguistic features and tackled data de-identification and class imbalance using Random Forest and XGBoost.
* **[Artificially Intelligent Game Using Q-Learning](https://github.com/hari047/Artificially-Intelligent-Game)** - An artificially intelligent game that performs reinforcement learning (Q-learning) to learn to how to navigate across (play) the 8x8 grid in the game.
* **[Predicting heart disease using machine learning](https://github.com/hari047/heart-disease-classification)** - Predicting heart disease using machine learning
* **[Botnet Detection Using ML](https://github.com/hari047/Botnet-Detection-Using-ML)** - A Botnet Detector that applies Deep Learning to assess a dataset and trains itself to identify the existence of bots in a network by analyzing the IRC traffic using the signatures of bots.
